# Vakit

Namaz vakitlerini takip ettiğiniz mobil uygulama. Artık konum izinleriyle birlikte manuel olarak takip etmek istediğiniz şehirleri de kaydedebilir ve farklı diller arasında geçiş yapabilirsiniz.

## Öne çıkan özellikler

- GPS konumunu yenileyip otomatik olarak takip etme
- Aynı anda en fazla üç manuel konumu kaydedip **Şehirler** panelinde farklı bir düzenle görüntüleme
- Türkçe, İngilizce ve Arapça arayüz desteği
- Şehir panelinde bir sonraki vakit ve kalan süreyi hızlıca kontrol etme, aktif konumu tek dokunuşla güncelleme

## Geliştirme

Projeyi çalıştırmak için temel adımlar:

```powershell
cd "c:\work\elemanlar\YAZILIMSAL PROJELER\flutter\namazvakitleri\Vakit"
flutter pub get
flutter gen-l10n
flutter run
```

Testleri çalıştırmak için:

```powershell
flutter test
```
